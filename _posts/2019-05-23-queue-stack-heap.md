|---------|----------|----------|--------------------------------------------------------------------------------------------------------------|
| queue   | file     | 队列     | First In First Out                                                                                           |
| stack   | pile     | 栈, 堆栈 | Last In First Out                                                                                            |
| heap    | tas      | 堆       | Tree-based data structure. In a heap, the highest (or lowest) priority element is always stored at the root. |
