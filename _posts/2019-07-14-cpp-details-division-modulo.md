```
cout << (-23) / 10 << endl;
-2
```

```
cout << (-23) % 10 << endl;
-3
```

```
cout << (-23) / (-10) << endl;
Out:
2
```

```
cout << (-23) % (-10) << endl;
Out:
-3
```
